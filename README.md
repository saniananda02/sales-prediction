# 📊 Sales Prediction using Machine Learning

## 📌 Project Overview
This project predicts product sales based on advertising spend across different platforms:
- TV
- Radio
- Newspaper

The goal is to understand how advertising impacts sales and build a machine learning model to predict future sales.

---

## 🚀 Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Google Colab

---

## 📂 Dataset
The dataset contains advertising budgets and corresponding sales.

Features:
- TV advertising spend
- Radio advertising spend
- Newspaper advertising spend

Target:
- Sales

---

## 🔍 Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Selection
5. Model Training (Linear Regression)
6. Model Evaluation
7. Prediction
